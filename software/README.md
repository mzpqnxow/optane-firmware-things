# Update Software

- https://www.intel.com/content/www/us/en/support/products/99743/memory-and-storage/intel-optane-memory/intel-optane-memory-series.html

## FUT

Use the FUT software to run updates from a bootable CD. OS agnostic. Recommended for all Intel branded drives

## Intelmas

Use `intelmas` for all sortsa things

## When to not use Intelmas / FUT

OEM drives typically need a special file or process to upgrade. See the README.md
